<span data-ttu-id="c2ba9-104">Microsoft Office 365 是一种订阅服务, 提供了您已熟悉的最新版本的 Office 桌面应用程序, 例如 Word、Excel 和 PowerPoint。</span><span class="sxs-lookup"><span data-stu-id="c2ba9-104">Microsoft Office 365 is a subscription service that provides you with the latest version of the Office desktop applications that you're already familiar with, such as Word, Excel, and PowerPoint.</span></span>
  
<span data-ttu-id="c2ba9-p101">即使 Office 365 是基于云的服务，Office 应用程序也不会在云中运行。但是，用户可以从 Office 365 门户下载 Office 应用程序并在其本地计算机上安装它们。Microsoft 使用即点即用技术，让 Office 应用程序的下载和安装既简单又迅速。即点即用使用虚拟化技术在本地计算机上的一个独立环境中运行 Office 应用程序，可让用户同时运行 Office 应用程序和较早版本的 Office。</span><span class="sxs-lookup"><span data-stu-id="c2ba9-p101">Even though Office 365 is a cloud-based service, Office applications don't run in the cloud. Microsoft uses Click-to-Run technology to make the download and installation of Office applications fast and simple. Click-to-Run uses virtualization technology to run Office applications in a self-contained environment on a local computer, which allows users to run Office applications side-by-side with earlier versions of Office.</span></span>
