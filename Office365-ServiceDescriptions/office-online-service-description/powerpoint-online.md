# <a name="powerpoint-online"></a>PowerPoint Online

PowerPoint Online (以前称为 PowerPoint Web App) 将 Microsoft PowerPoint 体验扩展到 Web 浏览器, 您可以直接在存储演示文稿的网站上处理演示文稿。 使用 Office Online 的 Microsoft Office 365 客户可以随时查看、创建和编辑文件。
  
The PowerPoint Editor is a web front-end component that creates a browser-based editing surface, which enables users to work on documents without losing fidelity.
  
如果单击 PowerPoint Online 工具栏上的 "**在 powerpoint 中打开**" 按钮, 演示文稿将在 powerpoint 桌面应用程序中打开 (如果计算机上安装了 Microsoft PowerPoint 2010 或更高版本)。 
  
了解如何[使用 Office 365 商业版从电脑下载和安装 Office](https://support.office.com/en-us/article/Download-and-install-Office-using-Office-365-for-business-on-a-PC-72977511-dfd1-4d8b-856f-405cfb76839c)。
  
## <a name="alignment-bullets-numbers"></a>Alignment, bullets, numbers
<a name="bkmk_AlignmentBulletsNumbers"> </a>

借助 PowerPoint Online，可以更改段落对齐方式、应用项目符号或编号，并更改带项目符号或编号的文本的级别。详细了解[在浏览器与 PowerPoint 桌面应用程序中使用演示文稿的区别](https://go.microsoft.com/fwlink/?LinkId=272763)。
  
## <a name="apply-basic-transitions-and-animations"></a>Apply basic transitions and animations
<a name="bkmk_Applybasictransitions"> </a>

从动画和切换效果库中进行选择。 PowerPoint Online 支持8个转换和37个动画。 PowerPoint Online 不支持的其他动画和转换将保留在演示文稿中并显示在幻灯片放映中, 但不能在 PowerPoint Online 中进行修改。 详细了解[在浏览器与 PowerPoint 桌面应用程序中使用演示文稿的区别](https://go.microsoft.com/fwlink/?LinkId=272763)。
  
## <a name="apply-rich-formatting-to-text-shapes-and-pictures"></a>Apply rich formatting to text, shapes, and pictures
<a name="bkmk_Applyrichformatting"> </a>

在 PowerPoint Online 中不可用。PowerPoint 桌面应用程序中提供了高级格式设置功能，如更多颜色、渐变、取色器、效果和样式。借助 PowerPoint Online，您可以添加文本框或从形状库中进行选择，并应用定义填充、边框和阴影效果的样式。 
  
## <a name="apply-themes-and-theme-variants"></a>Apply themes and theme variants
<a name="bkmk_ApplyThemes"> </a>

向演示文稿添加新幻灯片时，请从内置主题和变体库中进行选择。不受 PowerPoint Online 支持的主题保留在演示文稿中，但无法在 PowerPoint Online 中进行修改。详细了解[在浏览器与 PowerPoint 桌面应用程序中使用演示文稿的区别](https://go.microsoft.com/fwlink/?LinkId=272763)。
  
## <a name="office-add-ins"></a>Office 加载项
<a name="bkmk_AppsOffice"> </a>

PowerPoint Online 仅支持 Office 的内容外接程序。 内容外接程序将基于 web 的功能集成为可在演示文稿中显示的内容。 详细了解 [Office 外接程序类型](https://go.microsoft.com/fwlink/p/?LinkId=282310)。
  
## <a name="arrange-objects"></a>排列对象
<a name="bkmk_ArrangeObjects"> </a>

在 PowerPoint Online 中, 在图层中对形状和文本框进行移动、调整大小、旋转或排序。 对形状取消分组，以对其进行单独处理。 详细了解[在浏览器与 PowerPoint 桌面应用程序中使用演示文稿的区别](https://go.microsoft.com/fwlink/?LinkId=272763)。
  
## <a name="broadcast-slide-show"></a>Broadcast slide show
<a name="bkmk_BroadcastSlideShow"> </a>

需要有 PowerPoint 桌面应用程序，才能通过 PowerPoint Online 向远程受众广播幻灯片放映。无论是否安装了 PowerPoint 桌面应用程序，受众都可以通过 PowerPoint Online 现场观看演示文稿。详细了解[广播幻灯片放映](https://go.microsoft.com/fwlink/?LinkId=272765)。
  
## <a name="clipboard"></a>剪贴板
<a name="bkmk_Clipboard"> </a>

借助 PowerPoint Online，可以剪切、复制和粘贴演示文稿中的内容。用户可以在 Office Online 程序之间以及在 Office Online 和计算机上的 Microsoft Office 桌面应用程序之间复制和粘贴文本。详细了解[在浏览器与 PowerPoint 桌面应用程序中使用演示文稿的区别](https://go.microsoft.com/fwlink/?LinkId=272763)。
  
## <a name="create-and-manage-slides"></a>Create and manage slides
<a name="bkmk_CreateManageSlides"> </a>

在 PowerPoint Online 中对幻灯片执行添加、重新排序、复制、隐藏和删除操作。详细了解[在浏览器与 PowerPoint 桌面应用程序中使用演示文稿的区别](https://go.microsoft.com/fwlink/?LinkId=272763)。
  
## <a name="create-custom-animation"></a>Create custom animation
<a name="bkmk_Createcustomanimation"> </a>

PowerPoint Online 包括一个动画效果库。 不受 PowerPoint Online 支持的动画保留在演示文稿中，并会在幻灯片中显示，但无法在 PowerPoint Online 中进行修改。 必须在计算机上安装 PowerPoint 桌面应用程序，才能创建自定义动画。 详细了解如何[使用 PowerPoint 2013 桌面应用程序创建自定义动画](https://go.microsoft.com/fwlink/p/?LinkId=282348)。
  
## <a name="design-tools--advanced"></a>Design tools- advanced
<a name="bkmk_DesigntoolsAdvanced"> </a>

在 PowerPoint Online 中不可用。仅 PowerPoint 桌面应用程序提供了高级设计功能，如创建幻灯片母版和修改版式。借助 PowerPoint Online，您可以以自己的模板文件为基础创建新文件或从预定义的主题列表中进行选择。
  
## <a name="dropbox"></a>Dropbox
<a name="bkmk_Dropbox"> </a>

Dropbox 是托管提供云存储、文件同步、个人云和客户端软件的服务的文件。
  
## <a name="embed-presentation-on-web-or-blog-page"></a>在 Web 或博客页面上嵌入演示文稿
<a name="bkmk_Embedpresentation"> </a>

借助 PowerPoint Online，可以在网站或博客上嵌入演示文稿，以便任何人都可以查看你的信息，即使他们没有安装 Microsoft PowerPoint 桌面应用程序，也不例外。嵌入式查看器可显示动画、切换效果和音频/视频，与全保真阅读视图相同。详细了解如何[在网站或博客页面上嵌入演示文稿](https://go.microsoft.com/fwlink/?LinkId=272766)。
  
## <a name="font-formatting"></a>字体格式
<a name="bkmk_FontFormatting"> </a>

借助 PowerPoint Online，您可以将粗体、斜体、下划线、字体大小和颜色应用到文本。您还可以使用格式刷复制整个形状的格式。
  
## <a name="full-fidelity-reading-view"></a>Full fidelity reading view
<a name="bkmk_FullFidelityReading"> </a>

以预期方式查看演示文稿 - 显示动画、切换和音频/视频。
  
## <a name="full-ink-support"></a>完全墨迹支持
<a name="bkmk_FullInkSupport"> </a>

无法将墨迹插入到 PowerPoint Online 中，但 PowerPoint Online 会按照预期方式对其进行显示。
  
## <a name="full-selection-of-animations-and-transitions"></a>Full selection of animations and transitions
<a name="bkmk_FullAnimationsTransitions"> </a>

PowerPoint Online 支持8个转换和37个动画。 如需更多选择，请使用 PowerPoint 桌面应用程序。 
  
## <a name="headers-and-footers"></a>页眉和页脚
<a name="bkmk_HeadersFooters"> </a>

无法在 PowerPoint Online 中插入、编辑或删除页眉和页脚（包括日期和幻灯片编号），但 PowerPoint Online 会按预期显示页眉和页脚。详细了解[在浏览器与 PowerPoint 桌面应用程序中使用演示文稿的区别](https://go.microsoft.com/fwlink/?LinkId=272763)。
  
## <a name="hyperlinks"></a>Hyperlinks
<a name="bkmk_Hyperlinks"> </a>

插入、编辑和跟踪超链接。书签链接可正常运行，但不能在 PowerPoint Online 中进行编辑。
  
## <a name="insert-online-video"></a>插入联机视频
<a name="bkmk_InsertOnlineVideo"> </a>

视频和音频内容在阅读视图中和幻灯片放映模式下播放，文件大小限制为 100 MB。 可以从 YouTube 中插入在线视频, 并且可以在 PowerPoint Online 中调整媒体控件的大小、移动和删除。 但是, 若要插入除 YouTube 之外的音频和视频, 您需要 Microsoft PowerPoint 桌面应用程序。
  
## <a name="integration-with-excel-for-charts"></a>Integration with Excel for charts
<a name="bkmk_IntegrationExcel"> </a>

在 PowerPoint Online 中不可用。借助 PowerPoint Online，可以在现有演示文稿中查看 Excel 图表，但无法使用 PowerPoint Online 在演示文稿中编辑或插入 Excel 图表。详细了解如何[将 Excel 图表复制到 PowerPoint 中](https://go.microsoft.com/fwlink/?LinkId=272773)。
  
## <a name="navigation-slide-sorter"></a>Navigation— slide sorter
<a name="bkmk_Navigation"> </a>

在 PowerPoint Online 中不可用。PowerPoint Online 中未提供幻灯片浏览视图，该视图使您能够查看幻灯片缩略图，以便可以轻松地组织幻灯片并对其进行排序。利用 PowerPoint 桌面应用程序，您可以使用幻灯片浏览视图组织幻灯片，添加内容并将幻灯片分为不同类别。
  
## <a name="offline-viewing-and-authoring"></a>Offline viewing and authoring
<a name="bkmk_Offlineviewingandauthoring"> </a>

PowerPoint Online 从 Web 浏览器启动并且要依靠 Internet 连接。要脱机访问演示文稿，必须在计算机上安装 Microsoft PowerPoint 桌面应用程序，用于查看和编辑幻灯片。
  
## <a name="picture-cropping"></a>图片裁剪
<a name="bkmk_PictureCropping"> </a>

使用裁剪工具提高图片的主题帧。只需单击图片边缘的其中一个裁剪控点并拖动，直到获得所需的图片。
  
## <a name="pictures"></a>图片
<a name="bkmk_Pictures"> </a>

借助 PowerPoint Online，可以插入计算机中存储的图片，也可以插入必应图像中的图片。可以移动、调整和裁剪图片，并应用许多图片样式。PowerPoint Online 中未提供更复杂的图片处理功能，如应用效果。不能在 PowerPoint Online 中创建屏幕截图，但演示文稿中的屏幕截图可在 PowerPoint Online 中作为图片显示。详细了解[在浏览器与 PowerPoint 桌面应用程序中使用演示文稿的区别](https://go.microsoft.com/fwlink/?LinkId=272763)。
  
## <a name="present-online-through-skype-for-business-or-the-office-presentation-service"></a>Present online through Skype for Business or the Office Presentation Service
<a name="bkmk_PresentOnline"> </a>

在 PowerPoint Online 中不可用。借助 Microsoft PowerPoint 桌面应用程序，可以使用 Office Presentation Service 传递演示文稿，这是一项免费的公共服务，以便其他人能够继续使用 Web 浏览器。详细了解 [Office Presentation Service](https://go.microsoft.com/fwlink/?LinkId=272774)。
  
## <a name="presenter-view"></a>Presenter view
<a name="bkmk_PresenterView"> </a>

在 PowerPoint Online 中不可用。演示者仅可通过 Microsoft PowerPoint 桌面应用程序对演示文稿流、备注、批注和缩放工具进行幕后控制。PowerPoint Online 不支持这些功能。
  
## <a name="print-to-pdf"></a>打印为 PDF 格式
<a name="bkmk_PrinttoPDF"> </a>

借助 PowerPoint Online，可以将演示文稿打印输出到 PDF 阅读器中，其中幻灯片的所有版式和格式将按预期打印输出。详细了解[可以使用 PowerPoint Online 执行的基本任务](https://go.microsoft.com/fwlink/?LinkId=272775)。
  
## <a name="proofing-tools"></a>校对工具
<a name="bkmk_ProofingTools"> </a>

您可以使用 PowerPoint Online 的内置词典拼写检查并设置校对语言。但是，PowerPoint Online 不使用自定义词典，并且不包括翻译或同义词库。
  
## <a name="real-time-co-authoring"></a>实时共同创作
<a name="bkmk_Coauthoring"> </a>

多个作者可以同时在 PowerPoint Online 和 PowerPoint 2016 中工作。 实时状态可帮助您查看共同创作者在演示文稿中的编辑位置，从而不会在编辑时产生冲突，并且您可以在发生更改时进行查看。 PowerPoint 2013 支持同时编辑，但没有状态指示，并且在保存文档之前多个作者无法看到更改。 有关 PowerPoint 中的实时共同创作的详细信息, 请参阅[在 powerpoint 演示文稿](https://go.microsoft.com/fwlink/?linkid=844014)中协同工作。
  
## <a name="reviewer-tools---advanced"></a>审阅者工具-高级
<a name="bkmk_AdvancedReviewerTools"> </a>

在 PowerPoint Online 中不可用。 仅 PowerPoint 桌面应用程序中提供了高级审阅功能，如合并冲突和比较演示文稿。 借助 PowerPoint Online, 您可以查看、添加、编辑或删除注释。
  
## <a name="rights-management-apply-and-consume-irm-and-password-protection"></a>权限管理：应用和使用 IRM 和密码保护
<a name="bkmk_CreationPasswords"> </a>

在 PowerPoint Online 中不可用。PowerPoint Online 可显示使用信息权限管理 (IRM) 进行保护的演示文稿。但是，这些演示文稿无法在浏览器中进行编辑，您也无法在 PowerPoint Online 中创建 IRM 保护的演示文稿。PowerPoint Online 无法打开使用密码进行数字签名或加密的演示文稿。 
  
## <a name="run-slide-show"></a>Run slide show
<a name="bkmk_RunSlideShow"> </a>

借助 PowerPoint Online，仅使用 Web 浏览器和 Internet 连接即可运行演示文稿。详细了解[可以使用 PowerPoint Online 执行的基本任务](https://go.microsoft.com/fwlink/?LinkId=272775)。
  
## <a name="save-as-or-download-a-copy"></a>另存为或下载副本
<a name="bkmk_Save"> </a>

每当您做出更改时，PowerPoint Online 都会对您的工作进行保存。 没有"保存"命令。 您可以下载副本, 但必须具有 Microsoft PowerPoint 桌面应用程序才能编辑本地副本。
  
## <a name="shapes"></a>形状
<a name="bkmk_Shapes"> </a>

艺术字和图表都无法插入到 PowerPoint Online 中，但 PowerPoint Online 会按照预期方式显示它们。在"编辑"视图中，编辑文本并设置其格式。如果您希望将文本效果应用于艺术字，则需要使用 Microsoft PowerPoint 桌面应用程序。
  
## <a name="share"></a>共享
<a name="bkmk_Share"> </a>

如果将演示文稿保存在 SharePoint 文档库中，演示文稿会处于联机状态，可以通过发送链接（而非电子邮件附件）进行共享。拥有适当权限的人员可以在其 Web 浏览器或移动设备上查看演示文稿。详细了解如何[共享演示文稿](https://go.microsoft.com/fwlink/?LinkId=272776)。
  
## <a name="slide-show"></a>Slide show
<a name="bkmk_Slideshow"> </a>

PowerPoint Online 在全屏窗口中播放幻灯片放映。按下空格键可换片。可播放幻灯片动画，但仅支持对幻灯片使用淡化和擦除切换效果。详细了解[在浏览器与 PowerPoint 桌面应用程序中使用演示文稿的区别](https://go.microsoft.com/fwlink/?LinkId=272763)。
  
## <a name="smartart"></a>SmartArt
<a name="bkmk_SmartArt"> </a>

可以使用 PowerPoint Online 插入 SmartArt、切换到另一种版式或配色方案、应用 SmartArt 样式和编辑文本。详细了解[在浏览器与 PowerPoint 桌面应用程序中使用演示文稿的区别](https://go.microsoft.com/fwlink/?LinkId=272763)。
  
## <a name="table-creationediting-and-formatting"></a>Table creation/editing and formatting
<a name="bkmk_Tablecreation"> </a>

可以在 PowerPoint Online 中创建和编辑表格, PowerPoint Online 支持大多数表格功能。 对于高级功能 (如合并和拆分单元格), 请使用 PowerPoint 桌面应用程序。 详细了解[在浏览器与 PowerPoint 桌面应用程序中使用演示文稿的区别](https://go.microsoft.com/fwlink/?LinkId=272763)。
  
## <a name="tell-me"></a>操作说明搜索
<a name="Bkmk_TellMe_PPT"> </a>

当您需要在 PowerPoint Online 中完成一些处理事项，但又不知道该如何操作时，即可使用"告诉我"搜索功能来快速找到您寻找的答案。"告诉我"可以理解您要完成的事项，并通过给出建议的方式帮助您更快地做到这一点。
  
## <a name="undo-and-redo"></a>撤消和恢复
<a name="bkmk_Undoredo"> </a>

在活动演示文稿的当前编辑会话期间，可撤消 (Ctrl + Z) 和重做 (Ctrl + Y) 无限次最近的操作。如果编辑会话超时或切换到阅读视图的时间超过 30 秒，则会重置撤消历史记录。详细了解 [PowerPoint Online 中的键盘快捷键](https://go.microsoft.com/fwlink/?LinkId=272950)。
  
## <a name="view-and-add-comments"></a>查看和添加注释
<a name="bkmk_Viewaddcomments"> </a>

如果您的文件存储在 SharePoint Online 或 OneDrive for Business 上, 您可以添加、编辑或删除注释。 
  
如果您的文件存储在 OneDrive 上, 则可以在 "编辑" 视图中添加、编辑或删除注释, 还可以在阅读视图中查看和更新注释。
  
## <a name="view-and-edit-slide-notes"></a>View and edit slide notes
<a name="bkmk_Viewslidenotes"> </a>

借助 PowerPoint Online，可以显示或隐藏每张幻灯片的备注。您还可以在编辑模式下添加备注。
  
## <a name="wysiwyg-viewing"></a>WYSIWYG/所见即所得的查看
<a name="bkmk_WYSIWYGViewing"> </a>

打印或显示为已完成的产品时，以与其外观十分类似的格式编辑你的演示文稿。
  
## <a name="feature-availability"></a>功能可用性
<a name="bkmk_WYSIWYGViewing"> </a>

若要查看在各个 Office 365 计划、独立选项和内部部署解决方案之间的功能可用性，请参阅 [Office Online 服务说明](office-online-service-description.md)。
