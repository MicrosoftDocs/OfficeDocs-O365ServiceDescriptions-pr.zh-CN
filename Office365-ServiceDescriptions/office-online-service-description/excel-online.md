# <a name="excel-online"></a>Excel Online

Excel Online (以前称为 Excel Web App) Microsoft Excel 将您体验扩展到 web 浏览器，其中可以使用工作簿直接在存储工作簿的网站上。所有 Office 365 客户可以查看和宽松编辑使用 Office Online 的 Office 文件。
  
> [!NOTE]
> 无法在 Excel Online 中从 Microsoft SharePoint Online 中查看工作簿超过 30 兆字节 (MB)。若要查看较大的文件，您将需要在计算机上安装 Excel 桌面应用程序。 
  
> [!NOTE]
> 为帮助您比较提供的服务，本文中所列的高级功能包括只在 Office 套件中提供的 Excel 桌面应用程序功能（例如 Office Professional Plus、Office Standard 或 Office 365 ProPlus）。有关 Office Online 的所有功能（比如 Word Online 和 PowerPoint Online）的列表，请参阅 [跨 Office Online 计划的功能可用性](office-online-service-description.md#feature-availability-across-office-online-plans)
  
若要查看 Excel Online 支持哪种文件类型，请参阅 [Office Online 的受支持文件类型](office-online-service-description.md#supported-file-types-for-office-online)。要详细了解 Excel Online 和 Excel 桌面应用程序之间的差异，请参阅 [Differences between using a workbook in the browser and in Excel](https://go.microsoft.com/fwlink/?linkid=390875)（浏览器和 Excel 中工作簿的用法区别）。
  
## <a name="add-hyperlinks"></a>添加超链接
<a name="bkmk_Addhyperlinks"> </a>

通过在单元格中添加超链接，链接到网页或打开 Web 上的文件。您可以在单元格中直接键入 Web 地址，也可以使用"插入超链接"命令。
  
## <a name="advanced-time-filtering-timeline-slicer"></a>高级时间筛选（时间线切片器）
<a name="bkmk_Advancedtimefiltering"> </a>

时间线控制让你能够筛选模型化的数据。利用它可以通过可视方式查看和更改连续的日期范围，并筛选基于透视的对象（例如，数据透视表和数据透视图）。

<a name="bkmk_Coauthoring"> </a>

## <a name="tell-me"></a>告诉我
<a name="bkmk_Tell_Me_Excel"> </a>

当您需要在 Excel Online 中完成一些处理事项，但又不知道该如何操作时，即可使用"告诉我"搜索功能来快速找到您寻找的答案。"告诉我"可以理解您要完成的事项，并通过给出建议的方式帮助您更快地做到这一点。
  
## <a name="undo-and-redo"></a>撤消和恢复
<a name="bkmk_UndoRedo"> </a>

Excel Online 自动保存你的工作。如果你操作错误，可以使用"撤消"或按下 Ctrl + Z，也可以使用"恢复"或按下 Ctrl + Y。了解有关 [Excel Online 中的键盘快捷方式](https://go.microsoft.com/fwlink/p/?LinkId=272943)的详细信息。
  
## <a name="vba-and-macro-scripting"></a>VBA 和宏脚本
<a name="bkmk_VBAMacrosscripting"> </a>

Microsoft Excel 桌面应用程序是一款极其强大的工具，可用于操作、分析和显示数据。尽管它具有丰富的功能集，但有时贵组织会发现使用 Visual Basic for Applications (VBA) 更加简单。VBA 是一种编程语言，用于创建宏以执行单调、重复的任务，或执行用户界面 (UI) 似乎无法处理的某些任务。你无法在 Excel Online 中使用 VBA 创建宏，但可以打开并编辑启用 VBA 的电子表格，而无需删除（或破坏）文件中包含的 VBA。了解有关[宏入门](https://go.microsoft.com/fwlink/p/?LinkId=271678)的详细信息。
  
## <a name="view-3d-charts"></a>查看 3D 图表
<a name="bkmk_View3dcharts"> </a>

在您的工作表中查看 3D 图表。您无法在 Excel Online 中创建或编辑 3D 图表。
  
## <a name="what-if-analysis-tools"></a>What if 分析工具
<a name="bkmk_WhatIfAnalysis"> </a>

在 Excel Online 中不可用。你可以按预期那样在 Excel Online 中查看"如果"情形的分析结果。如果你要使用分析工具（例如"单变量求解"、"数据表"、"规划求解"和"系列"），则将需要 Excel 桌面应用程序。了解有关[浏览器和 Excel 桌面应用程序中工作簿用法的差异](https://go.microsoft.com/fwlink/p/?LinkId=273900)的详细信息。
