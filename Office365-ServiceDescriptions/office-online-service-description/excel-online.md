# <a name="excel-online"></a>Excel Online

Excel Online (以前称为 Excel Web App) 将您的 Microsoft Excel 体验扩展到 Web 浏览器, 您可以直接在存储工作簿的网站上处理工作簿。 所有 Office 365 客户都可以使用 Office Online 查看并简单编辑 Office 文件。
  
> [!NOTE]
> 在 Microsoft SharePoint Online 中, 不能在 Excel Online 中查看超过 30 mb 的工作簿。 要查看较大的文件，需要在您的计算机上安装 Excel 桌面应用程序。 
  
> [!NOTE]
> 为帮助您比较提供的服务，本文中所列的高级功能包括只在 Office 套件中提供的 Excel 桌面应用程序功能（例如 Office Professional Plus、Office Standard 或 Office 365 ProPlus）。 有关所有 Office Online 功能 (如 Word Online 和 PowerPoint Online) 的列表, 请参阅[跨 Office Online 计划的功能可用性](https://docs.microsoft.com/office365/servicedescriptions/office-online-service-description/office-online-service-description)。 

若要查看 Excel Online 支持哪种文件类型，请参阅 [Office Online 的受支持文件类型](https://docs.microsoft.com/office365/servicedescriptions/office-online-service-description/office-online-service-description)。要详细了解 Excel Online 和 Excel 桌面应用程序之间的差异，请参阅 [Differences between using a workbook in the browser and in Excel](https://support.office.com/article/differences-between-using-a-workbook-in-the-browser-and-in-excel-f0dc28ed-b85d-4e1d-be6d-5878005db3b6?ocmsassetID=HA010369179&CorrelationId=1b10d00b-d045-439b-a3e4-b03be889090e&ui=en-US&rs=en-US&ad=US)（浏览器和 Excel 中工作簿的用法区别）。

在 Excel Online 中不可用。 如果你在计算机上安装了 Excel 桌面应用程序，则可以使用 Excel 查找并替换文档中的内容。 了解有关[在 Excel 中使用查找和替换的](https://go.microsoft.com/fwlink/p/?LinkId=282333)详细信息。
  
## <a name="rights-management-irm-and-password-security"></a>Rights Management: IRM and password security
<a name="bkmk_RightsManagement"> </a>

 在 Excel 桌面应用程序中使用密码、权限和其他限制保护工作簿。在 Excel Online 中，你可以加载并与包含工作表保护（防止用户在受保护的单元格中进行选择或键入）的工作簿进行交互。了解有关[保护 Excel 工作簿](https://go.microsoft.com/fwlink/p/?LinkId=271672)的详细信息。
  
## <a name="save-or-download-a-copy"></a>Save or Download a copy
<a name="bkmk_SaveDownload"> </a>

是否想要自己的工作簿副本？使用"文件"选项卡上的"另存为"命令，或右键单击其文件夹中的工作簿名称，使用"下载"命令将副本发送到您的计算机。
  
## <a name="share"></a>共享
<a name="bkmk_Share"> </a>

单击一下即可发送链接，其他人可以使用该链接查看或编辑文档。了解有关[使用 Office Online 在 Office 365 中协同工作](https://go.microsoft.com/fwlink/p/?linkid=272770)的详细信息。
  
## <a name="sheet-protection"></a>工作表保护
<a name="bkmk_SheetProtection"> </a>

当你与其他用户共享一个 Excel 文件时，你可以保护工作表，以防止其被更改，并且你可以选择允许工作表用户执行的操作。
  
## <a name="slicer-viewing"></a>切片器查看
<a name="bkmk_Slicerviewing"> </a>

查看您工作表中的切片器，通过单击切片器按钮筛选数据。您无法在 Excel Online 中创建或编辑切片器。
  
## <a name="sort-and-filter-data"></a>Sort and filter data
<a name="bkmk_Sortfilterdata"> </a>

在 Excel Online 中，您可以查看和重新排序所有工作表、排序和筛选数据，并深入了解 PivotTables 的详细信息。
  
## <a name="spreadsheet-audit-and-compliance"></a>Spreadsheet audit and compliance
<a name="bkmk_Spreadsheetauditcompliance"> </a>

在 Microsoft Excel 桌面应用程序中，你可以使用工作表查询和比较功能来比较工作簿的版本、分析工作簿中的问题或不一致，或查看工作簿与工作表之间的链接。Excel Online 不支持这些功能。了解有关[你可以用电子表格查询功能来做些什么](https://go.microsoft.com/fwlink/p/?LinkId=271677)的详细信息。
  
## <a name="status-bar-aggregates"></a>Status bar aggregates
<a name="bkmk_StatusBarAggregates"> </a>

当您在 Excel Online 中选择一组单元格后，会在状态栏中看到"SUM"、"AVERAGE"和"COUNT"。您还可以通过选择要查看的聚合，对状态栏进行自定义。
  
## <a name="surveys"></a>调查
<a name="bkmk_Surveys"> </a>

使用调查向用户发送一个他们可以填写的 Excel Online 表格。系统会自动对在线工作表中回复的内容进行编译。了解有关 [Excel 中的调查](https://go.microsoft.com/fwlink/p/?LinkId=391470)的详细信息。
  
## <a name="tell-me"></a>操作说明搜索
<a name="bkmk_Tell_Me_Excel"> </a>

当您需要在 Excel Online 中完成一些处理事项，但又不知道该如何操作时，即可使用"告诉我"搜索功能来快速找到您寻找的答案。"告诉我"可以理解您要完成的事项，并通过给出建议的方式帮助您更快地做到这一点。

<a name="bkmk_Toaldata"></a>您可以通过单击 "开始" 选项卡中 "格式为表格" 下的 "切换汇总行" 来快速汇总表中的数据。 

  
## <a name="undo-and-redo"></a>撤消和恢复
<a name="bkmk_UndoRedo"> </a>

Excel Online 自动保存你的工作。如果你操作错误，可以使用"撤消"或按下 Ctrl + Z，也可以使用"恢复"或按下 Ctrl + Y。了解有关 [Excel Online 中的键盘快捷方式](https://go.microsoft.com/fwlink/p/?LinkId=272943)的详细信息。
  
## <a name="vba-and-macro-scripting"></a>VBA and macro scripting
<a name="bkmk_VBAMacrosscripting"> </a>

Microsoft Excel 桌面应用程序是一款极其强大的工具，可用于操作、分析和显示数据。尽管它具有丰富的功能集，但有时贵组织会发现使用 Visual Basic for Applications (VBA) 更加简单。VBA 是一种编程语言，用于创建宏以执行单调、重复的任务，或执行用户界面 (UI) 似乎无法处理的某些任务。你无法在 Excel Online 中使用 VBA 创建宏，但可以打开并编辑启用 VBA 的电子表格，而无需删除（或破坏）文件中包含的 VBA。了解有关[宏入门](https://go.microsoft.com/fwlink/p/?LinkId=271678)的详细信息。
  
## <a name="view-3d-charts"></a>View 3D charts
<a name="bkmk_View3dcharts"> </a>

在您的工作表中查看 3D 图表。您无法在 Excel Online 中创建或编辑 3D 图表。
