# <a name="onenote-online"></a>OneNote Online

使用 OneNote 联机 (以前称为 OneNote Web App) 可从任意位置添加到和轻松地与其他人共享的 OneNote 笔记本中做笔记联机。所有 Office 365 客户可以查看和宽松编辑使用 Office Online 的 Office 文件。 
  
## <a name="advanced-collaboration-new-content-shows-as-unread-presence"></a>高级协作：新内容显示为未读状态
<a name="bkmk_AdvancedCollaboration"> </a>

在 OneNote Online 中不可用。通过 OneNote 桌面应用程序，您可以使用 OneNote 中的集成配置文件更轻松地识别其他作者并对其进行身份验证。按作者的姓名搜索笔记本更改和修订，并在返回到共享笔记本时查看最近的所有编辑。
  
## <a name="apply-tags"></a>应用标记
<a name="bkmk_Tags"> </a>

通过 OneNote Online，可以对笔记应用各种标记，以便轻松进行整理和跟进。例如，将笔记标记为问题、待办事项或联系信息。详细了解如何[浏览器和 OneNote 桌面应用程序中笔记本用法的差异](https://go.microsoft.com/fwlink/p/?LinkId=272946)。
  
## <a name="office-add-ins"></a>Office 外接程序
<a name="bkmk_AppsforOffice"> </a>

OneNote Online 仅支持 Office 的内容外接程序。内容外接程序会将基于 Web 的功能集成为可配合文档显示的内容。详细了解 [Office 外接程序类型](https://go.microsoft.com/fwlink/p/?LinkId=282310)。
  
## <a name="audio-notes-recording"></a>音频笔记录音
<a name="bkmk_AudioVideoLinked"> </a>

在 OneNote Online 中不可用。通过 OneNote 桌面应用程序，您可以录制音频和视频笔记，可在录制时将其直接链接到记录的任何文本笔记。OneNote Online 不会播放音频和视频内容，但是此介质将保留在笔记本中，可将这些音频和视频文件下载到您的计算机上进行播放。 
  
## <a name="bullets-and-numbering"></a>项目符号和编号
<a name="bkmk_BulletsNumbering"> </a>

通过 OneNote Online，可以选择是应用三种项目符号样式，还是应用五种编号样式。单击"增加缩进"和"减少缩进"按钮可更改笔记本以及 OneNote Online 中创建的现有项目符号列表和编号列表的列表级别。详细了解[浏览器和 OneNote 桌面应用程序中笔记本用法的差异](https://go.microsoft.com/fwlink/p/?LinkId=272946)。
  
## <a name="clipboard"></a>剪贴板
<a name="bkmk_Clipboard"> </a>

通过 OneNote Online，可以在笔记本中剪切、复制和粘贴内容。可以在 OneNote Online 和 OneNote 桌面应用程序之间复制和粘贴文本。详细了解[浏览器和 OneNote 桌面应用程序中笔记本用法的差异](https://go.microsoft.com/fwlink/p/?LinkId=272946)。
  
## <a name="clipping-experiences"></a>剪辑体验
<a name="bkmk_ClippingExperiences"> </a>

在 OneNote Online 中不可用。通过 OneNote 桌面应用程序，您可以剪辑屏幕上显示的任何内容、向笔记本分区发送网页或整个文档，或者记下可作为笔记本的一部分自动保存和归档的快速笔记。
  
## <a name="co-authoring"></a>共同创作
<a name="bkmk_Coauthoring"> </a>

通过 OneNote Online，可以与在其他位置使用 OneNote Online 或 OneNote 桌面应用程序的人员同时编辑笔记本。详细了解如何[共同使用 OneNote 笔记本](https://go.microsoft.com/fwlink/p/?LinkId=272945)。
  
## <a name="create-and-manage-pages-and-sections"></a>创建和管理页面和分区
<a name="bkmk_CreateManagePagesSections"> </a>

通过 OneNote Online，可以在分区中创建新页面，或者在笔记本中创建新分区，从而轻松向新笔记本或现有笔记本添加内容。可以在笔记本之间自定义、格式化、移动或复制页面和分区。详细了解如何[使用 OneNote Online 笔记本](https://go.microsoft.com/fwlink/p/?LinkId=272945)。
  
## <a name="download-copy-of-embedded-office-files"></a>下载嵌入式 Office 文件的副本
<a name="bkmk_DownloadCopyEmbeddedOfficeFiles"> </a>

通过 OneNote Online，可以插入 Office 文件作为附件或打印输出，以将其存储为笔记本的一部分。OneNote 2013 桌面应用程序新增了在笔记中嵌入 Excel 电子表格和 Visio 绘图的功能。详细了解如何[在 OneNote 中嵌入 Microsoft Office 文件](https://go.microsoft.com/fwlink/p/?LinkId=271843)。
  
## <a name="dropbox"></a>Dropbox
<a name="bkmk_Dropbox"> </a>

Dropbox 是托管提供云存储、文件同步、个人云和客户端软件的服务的文件。
  
## <a name="edit-embedded-files"></a>编辑嵌入的文件
<a name="EmbeddedFileEdit"> </a>

通过 OneNote Online，您可以将嵌入的文件下载到计算机以便打开。通过 OneNote 桌面应用程序，您可以将任何计算机文件附加到笔记的任何部分，这样即在笔记本中存储了文件的副本。甚至可以将 Outlook 会议详细信息和 Outlook 任务插入到 OneNote，或者将 OneNote 页面通过电子邮件发送到 Outlook。此外，还可以在 OneNote 中创建或导入 Excel 电子数据表和 Visio 图表，并在笔记中适当编辑其信息。插入的文件在笔记页上显示为图标。双击任意图标可打开其文件。 
  
## <a name="equations"></a>方程
<a name="bkmk_Equations"> </a>

在 OneNote Online 中不可用。通过 OneNote 桌面应用程序，您可以在会议、讨论会或课堂上记下数学方程，OneNote 可以立即为您计算结果。
  
## <a name="hyperlinks"></a>超链接
<a name="bkmk_hyperlinks"> </a>

通过 OneNote Online，可以向 Web 地址插入超链接或对选定文本应用链接。还可以右键单击页面选项卡，复制相应页面的直接链接。详细了解[浏览器和 OneNote 桌面应用程序中笔记本用法的差异](https://go.microsoft.com/fwlink/p/?LinkId=272946)。
  
## <a name="image-optical-character-recognition-ocr"></a>图像光学字符识别 (OCR)
<a name="bkmk_ImageOCR"> </a>

在 OneNote Online 中不可用。光学字符识别 (OCR) 可将文本图像（例如扫描的文档）转换为仅在 OneNote 桌面应用程序中可识别的实际文本字符。 
  
## <a name="ink-viewing"></a>墨迹查看
<a name="bkmk_InkViewing"> </a>

在 OneDrive 中，OneNote Online 显示墨迹，但是公式显示为占位符。在 SharePoint 中，墨迹可见，方程显示为占位符，而形状是隐藏的。这些不能插入到 OneNote Online 中，也不能在其中进行编辑。在 OneDrive 中，您可以选择并删除形状、墨迹和公式占位符。在 SharePoint 中，您可以选择并删除这些对象的占位符。
  
## <a name="inking-ink-to-text-and-math-customizable-pens-drawing-tools"></a>墨迹：文本着色和数学、可自定义笔、绘图工具
<a name="bkmk_Inking"> </a>

在 OneNote Online 中不可用。通过 OneNote 桌面应用程序，您可以用手指、触笔或鼠标顺利地绘制、擦除和编辑。如果您更喜欢书写而不是打字，OneNote 可以切换至手写。如果从 SharePoint 文档库打开笔记本，笔记本中的任何方程和墨迹均显示为占位符。 
  
## <a name="linked-notes"></a>链接的笔记
<a name="bkmk_LinkedNotes"> </a>

向文件链接笔记记录的功能在 OneNote Online 中不可用，并且指向文件的链接是隐藏的。但是，这些链接仍保留在笔记中，可以使用 OneNote 桌面应用程序打开。 
  
## <a name="navigation"></a>导航
<a name="bkmk_navigation"> </a>

通过 OneNote Online，您可以使用快速访问工具栏中浏览器样式的"后退"和"前进"按钮实现在页面之间快速跳转。
  
## <a name="notebook-management-cross-sectionnotebook-page-filing-section-re-order-createdelete-section-groups"></a>笔记本管理：跨分区/笔记本页面归档、分区重新排序、创建/删除分区组
<a name="bkmk_AdvancedNotebookManagement"> </a>

在 OneNote Online 中不可用。高级管理功能（例如分区重新排序、创建和删除分区组以及拖放页面归档）仅在 OneNote 桌面应用程序中可用。 
  
## <a name="offline-viewing-and-authoring"></a>脱机查看和创作
<a name="bkmk_Offlineviewing"> </a>

在 OneNote Online 中不可用。Office Online 需要 Internet 连接和 Web 浏览器。需要在计算机上安装 OneNote 桌面应用程序才能在与 Office Web Apps Server 或 Internet 断开连接时查看和编辑文档。
  
## <a name="outlook-integration-tasks"></a>Outlook 集成（任务）
<a name="bkmk_OutlookIntegration"> </a>

OneNote Online 不支持与 Microsoft Outlook（例如电子邮件页面、Outlook 任务或会议详细信息）一起使用的命令。
  
## <a name="paragraph-formatting"></a>段落格式
<a name="bkmk_Paragraphformatting"> </a>

通过 OneNote Online，可以更改段落的对齐方式、增加或减小左边距缩进，也可以更改文字方向（从右到左或从左到右）。详细了解[浏览器和 OneNote 桌面应用程序中笔记本用法的差异](https://go.microsoft.com/fwlink/p/?LinkId=272946)。
  
## <a name="pictures"></a>图片
<a name="bkmk_Pictures"> </a>

通过 OneNote Online，可以通过文件或必应图像插入图片。还可以调整图片大小并添加可选文字。若要使用更多高级图片功能（如屏幕剪辑、扫描图像或联机图片），需要使用 OneNote 桌面应用程序。详细了解[浏览器和 OneNote 桌面应用程序中笔记本用法的差异](https://go.microsoft.com/fwlink/p/?LinkId=272946)。
  
## <a name="print"></a>打印
<a name="BKMK_Print_OneNote"> </a>

您可以在 OneNote Online 中打印备注。
  
## <a name="proofing-tools"></a>校对工具
<a name="bkmk_ProofingTools"> </a>

OneNote Online 可在键入时自动检查拼写，并在拼写错误的文本下标注红色波浪下划线。也包括常用的自动更正操作，如更正例程拼写错误或将字符转换成符号。此外，还可以设置校对语言或对所选文本关闭拼写检查。详细了解[浏览器和 OneNote 桌面应用程序中笔记本用法的差异](https://go.microsoft.com/fwlink/p/?LinkId=272946)。
  
## <a name="rights-management-apply-and-consume-irm-and-password-protection"></a>权限管理：应用和使用 IRM 和密码保护
<a name="bkmk_RightsManagement"> </a>

OneNote Online 显示受信息权限管理 (IRM) 保护的笔记本。但是，无法在浏览器中编辑这些笔记本，并且无法在 OneNote Online 中创建受 IRM 保护的笔记本。OneNote Online 无法打开使用密码加密的文档。高级文档保护功能（例如创建受 IRM 保护的笔记本和应用密码保护）仅在 OneNote 桌面应用程序中可用。
  
## <a name="search-on-page-within-sections"></a>搜索（在页面上，在分区内）
<a name="bkmk_SectionsOnpage"> </a>

通过 OneNote Online，可以使用"即时搜索"重新调用曾在 OneNote 中创建或保存的任何内容。详细了解[浏览器和 OneNote 桌面应用程序中笔记本用法的差异](https://go.microsoft.com/fwlink/p/?LinkId=272946)。
  
## <a name="search-by-tag-title-authordate-audio-notes"></a>搜索条件：标记、标题、作者/日期、音频笔记
<a name="bkmk_SearchTags"> </a>

在 OneNote Online 中不可用。高级搜索功能（例如按标记、标题、作者/日期和音频文件搜索）仅在 OneNote 桌面应用程序中可用。
  
## <a name="share"></a>共享
<a name="bkmk_Share"> </a>

如果已将 OneNote 笔记本保存到 SharePoint 文档库，则 OneNote 笔记本处于联机状态。也就是说，可以通过发送链接进行共享，而无需使用电子邮件附件。通过单击此链接，用户可在其 Web 浏览器中阅读你的笔记。详细了解如何[联机共享 OneNote 笔记](https://go.microsoft.com/fwlink/p/?LinkID=272945)。
  
## <a name="show-or-hide-authors"></a>显示或隐藏作者
<a name="bkmk_ShowHideAuthors"> </a>

通过 OneNote Online，您可以关闭某人编辑笔记本时显示的作者标记。这将删除页面上新笔记旁边显示的缩写。默认情况下启用作者标记。
  
## <a name="styles"></a>样式
<a name="bkmk_Styles"> </a>

通过 OneNote Online，可以轻松应用文本样式，以便快速设置格式。详细了解[浏览器和 OneNote 桌面应用程序中笔记本用法的差异](https://go.microsoft.com/fwlink/p/?LinkId=272946)。
  
## <a name="tables"></a>表
<a name="bkmk_Tables"> </a>

通过 OneNote Online，可以插入表格、编辑表格文本并轻松编辑基本表格结构，例如添加或删除行和列。若要使用更多高级表格功能（如将表格转换成 Excel 电子表格，或在表格单元格中使用单元格底纹、标题行和数据排序），需要使用 OneNote 桌面应用程序。详细了解[浏览器和 OneNote 桌面应用程序中笔记本用法的差异](https://go.microsoft.com/fwlink/p/?LinkId=272946)。
  
## <a name="template-support"></a>模板支持
<a name="bkmk_TemplateSupport"> </a>

在 OneNote Online 中不可用。通过 OneNote 桌面应用程序，您可以将模板用作可应用到笔记本中新页面的页面设计，从而为其提供更具吸引力的背景、更统一的外观或一致的布局。
  
## <a name="undo-and-redo"></a>撤消和恢复
<a name="bkmk_UndoRedo"> </a>

使用键盘可撤消 (Ctrl+ Z) 或重做 (Alt + F7) 在 OneNote Online 的活动笔记本中编辑的每个页面的最近操作。OneNote Online 提供了活动笔记本中编辑的每个页面的单独撤消历史记录。在当前编辑会话过程中，可对每个页面撤消无数次操作，直至插入图片或其他作者进行编辑时为止。移动和删除页面不能撤消。详细了解如 [OneNote Online 键盘快捷方式](https://go.microsoft.com/fwlink/p/?LinkId=272944)。
  
## <a name="video-playback"></a>视频播放
<a name="bkmk_Videoplayback"> </a>

视频笔记会保留，但无法在 OneNote Online 中录制。可以将视频文件下载到计算机进行播放。详细了解[浏览器和 OneNote 桌面应用程序中笔记本用法的差异](https://go.microsoft.com/fwlink/p/?LinkId=272946)。
  
## <a name="view-previous-page-versions"></a>查看以前的页面版本
<a name="bkmk_Viewpreviouspage"> </a>

通过 OneNote Online，可以查看并还原旧版页面，包括作者和创建时间。相对于旧版页面的更改会自动突出显示。详细了解[浏览器和 OneNote 桌面应用程序中笔记本用法的差异](https://go.microsoft.com/fwlink/p/?LinkId=272946)。
  
## <a name="feature-availability"></a>功能可用性
<a name="bkmk_Viewpreviouspage"> </a>

若要查看在各个 Office 365 计划、独立选项和内部部署解决方案之间的功能可用性，请参阅 [Office Online 服务说明](office-online-service-description.md)。
