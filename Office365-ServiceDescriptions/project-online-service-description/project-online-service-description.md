# <a name="project-online-service-description"></a>Project Online 服务说明

Project Online 是项目组合管理 (PPM) 和日常工作的一个灵活的联机解决方案。通过 Office 365 送达，Project Online 提供强大的项目管理功能的规划、 确定优先级和管理项目和项目组合投资 — 从几乎任何位置几乎任何设备上。Project Online 可由管理员、 项目组合经理和查看器、 项目和资源经理和工作组负责人和成员。 
  
Project Online 提供三个计划 - Project Online 协作版、Project Online Professional 和 Project Online 高级版。
  
若要比较这些计划的功能，请参阅[比较 Project Online 计划](https://products.office.com/en-us/project/compare-microsoft-project-management-software)。
  
您可以导出、保存以及打印 Office 365 服务说明中的页面。了解如何[导出多个页面](https://go.microsoft.com/fwlink/?LinkId=403349)。
  
## <a name="project-online-boundaries-and-limits"></a>Project Online 边界和限制

Project Online 服务存在一些限制。有关详细信息，请参阅 [Project Online 软件边界和限制](https://support.office.com/en-us/article/Project-Online-software-boundaries-and-limits-5A09DBCE-1E68-4A7B-B099-D5F1B21BA489)。
  
## <a name="data-backup"></a>数据备份

 Project Online 构建于 SharePoint Online 上，并且遵循与 SharePoint 相同的数据备份策略。有关详细信息，请参阅 [SharePoint Online 服务说明](https://technet.microsoft.com/en-us/library/sharepoint-online-service-description.aspx)中的数据备份一节。
  
## <a name="use-of-sharepoint-online"></a>使用 SharePoint Online

Project Online 服务需要使用 SharePoint Online，其中将设置为您的 Project Online 服务的一部分。Project Online Professional 或者 Premium 与提供的 SharePoint Online 功能的权限被限制为存储和访问数据以支持 Project Online。
  
## <a name="licensing-considerations"></a>许可注意事项

### <a name="internal-users"></a>内部用户

Project Online 网站上的任何交互需要 Project Online 的许可证。
  
- Project Online 协作版是最低要求。
    
- 有关更高级的使用情况下，用户需要有相应的许可证。项目经理至少需要 Project Online Professional 许可证。项目组合经理需要 Project Online Premium 许可证。
    
> [!IMPORTANT]
>  Project Online Essentials 是具有 Project Online Professional 或 Project Online Premium 的客户的团队成员*加载项*许可证。<br/>时您最近的 Project Online Professional 或 Project Online Premium 许可证到期，则将 120 天后删除 Project Web App 实例。<br/>Project Online 试用版订阅的 30 天试用期间结束后将被删除 Project Web App 实例。 
  
### <a name="external-users"></a>外部用户

Project Online 的外部用户授权遵循的用于内部用户相同的策略。Project Online 网站上的任何交互需要 Project Online 的许可证。
