# <a name="microsoft-project-service-description"></a>Microsoft 项目服务说明

## <a name="overview"></a>概述

Microsoft Project 提供了以下应用程序，以帮助满足组织对项目和工作管理的需求：

- Project 网页版
- Project Online 
- Project Online 桌面客户端

本文将帮助您了解每个项目订阅中提供了哪些应用程序，以及每个应用程序提供了哪些功能。

### <a name="project-for-the-web"></a>Project 网页版

针对基于云的工作和项目管理，web 的 Project 是 Microsoft 的最新产品。 Web 项目为满足大多数需求和角色提供了简单、强大的工作管理功能。 项目经理和工作组成员可以使用 web 项目来规划和管理任何规模的工作。

Web 的项目是基于 Microsoft Power Platform 建立的。 电源平台由 PowerApps、电源自动化、Power BI 和通用数据服务（CD）组成。 Web 数据的项目存储在 CD 中。 

可通过三个订阅（项目计划1、项目计划3和项目计划5）获取 web 的项目。 " [Web 功能的项目](#project-for-the-web-features)" 部分中列出的功能仅与 web 数据存储的项目中的数据进行交互。

### <a name="project-online"></a>Project Online
 
Project Online 是用于项目组合管理（PPM）和日常工作的灵活联机解决方案。 Project Online 提供强大的项目管理功能，以便在几乎任何设备上的几乎任何位置&mdash;规划、确定项目和项目组合投资的优先级并对其进行管理。 Project Online 适用于管理员、项目组合经理、项目组合查看者、项目经理、资源经理、工作组领导和成员。

Project Online 在 SharePoint 平台上构建，并将数据存储在 SharePoint 数据存储中。

Project Online 可通过以下两项订阅进行：项目计划3和项目计划5。 [Project online 功能](#project-online-features)部分中列出的功能仅与 Project online SharePoint 数据存储中的数据进行交互。

Project Online 的工作组成员功能通过 Project Online Essentials 订阅提供。 Project Online Essentials 是针对具有项目计划3或项目计划5订阅的客户的工作组成员 *附加* 订阅。 它为团队成员提供了一个 web 界面，用于对 Project Online 中管理的项目执行任务和时间更新。

具有 Project Online Essentials 的团队成员可以执行以下 Project Online 函数：

- 使用基于 web 的界面
- 更新任务、问题和风险
- 提交时间表
- 共享文档并与 Microsoft 团队或 Skype for Business 用户协作

Project Online Essentials 不包含 web 的项目。

### <a name="project-online-desktop-client"></a>Project Online 桌面客户端

Microsoft Project Online 桌面客户端是一个项目管理程序，可提供自动化的日程安排、项目资源管理和内置报告功能。 Project Online 桌面客户端可用作独立的应用程序，也可以连接到 Project Online。 Project Online 桌面客户端是 Project Professional 桌面客户端的订阅版本，可在项目计划3和项目计划5订阅中使用。

### <a name="using-project-for-the-web-and-project-online-together"></a>结合使用 web 项目和 Project Online

尽管 Project for web 和 Project Online 是独立的应用程序，但您可以并排使用它们。 您可以在项目主页中查看这两个应用程序中的所有项目。 您可以使用项目路线图功能构建包含来自这两个应用程序的项目的路线图。 您可以使用 Power BI 和 Project for web Power BI 内容包获取跨两个应用程序的项目和资源的见解。<sup>3</sup>

此图显示了应用程序可以并排协作的方式。

![在 web 和 Project Online 上一起显示项目的关系图](../media/project-online-applications.png)

今后，Microsoft 将重点关注 web 创新的项目。 不过，你可以继续使用 Project Online，因为它将继续获得关键性能和安全性改进。

## <a name="microsoft-project-subscriptions"></a>Microsoft Project 订阅

Microsoft Project 可通过三种不同的订阅使用，以最大限度地满足组织的需求。 这些订阅是项目计划1、项目计划3和项目计划5。 若要在各个订阅之间比较功能，请参阅[跨应用程序和订阅的功能可用性](#feature-availability-across-applications-and-subscriptions)。

|产品 | 计划 1 | 计划3 | 计划5 |
|--------|-----------------------------|------------------------|------------------------|
|Project 网页版 |**x**  | **x x** | **x x** |
|Project Online | |**x x**| **x x x** |
|Project Online 桌面客户端 | |**x**|**x**|

在上表中，检查次数（**x**）指示计划提供的常规级别的功能。 例如，对于 Project Online，计划5提供更多功能，然后计划3。

### <a name="licensing-considerations"></a>许可注意事项

- 在 Project Online 网站上进行任何交互都需要项目计划3或项目计划5订阅。
- 当您的最后一个项目计划1、项目计划3或项目计划5订阅过期时，将在120天后删除 web 实例的项目。
- 当您的最后一个项目计划3或项目计划5订阅过期时，Project Online 实例将在120天后删除。
- 对于 web 和 Project Online 试用订阅的 Project，在试用期结束后30天内将删除试用实例。

### <a name="use-of-sharepoint-online"></a>使用 SharePoint Online

Project Online 要求使用 SharePoint Online，这是作为 Project Online 的一部分进行设置的。 与项目计划3或项目计划5订阅一起提供的 SharePoint Online 功能的权限仅限于存储和访问数据以支持 Project Online。

### <a name="project-roadmap-and-power-automate"></a>项目路线图和电源自动化

项目路线图要求使用 Power 自动功能，这是作为项目订阅的一部分进行预配的。 对电源自动化功能的权限仅限于项目路线图所需的电源自动功能。 在 Microsoft 365 管理中心的 "**应用程序**" 部分中，"项目路线图" 所需的电力自动化功能显示为 "具有流服务计划的**项目的数据集成**"。

### <a name="project-for-the-web-and-common-data-service"></a>Web 和常见数据服务的项目

Web 的项目要求使用通用数据服务（CD）存储其数据。 将 CD 作为项目订阅的一部分进行预配。 对 CD 功能的权限仅限于存储和访问数据以支持 web 的 Project。 Project 所需的 CD 功能在 Microsoft 365 管理中心的 "**应用程序**" 部分显示为 project service Plan 的**通用数据服务**。

### <a name="data-backup-and-retention"></a>数据备份和保留

Web 和 Project Online 的项目与 Office 365 具有相同的数据备份和保留策略。 有关详细信息，请参阅[Office 365 中的数据保留、删除和销毁](https://docs.microsoft.com/office365/Enterprise/office-365-data-retention-deletion-and-destruction-overview)。
