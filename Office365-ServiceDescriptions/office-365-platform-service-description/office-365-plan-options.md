#### <a name="basic-client-limitations"></a>基本客户端限制

以下功能在完整客户端中可用，但在基本客户端中不可用：

- 管理团队呼叫设置
- 管理代理    
- 代表其他联系人发起呼叫（经理/代理方案）    
- 处理其他人的呼叫（如果配置为代理的话）    
- 管理大量呼叫    
- 向响应组发起呼叫
- 呼叫寄存
- 群呼代接
    
### <a name="changing-or-mixing-plans"></a>更改或混合计划

随着组织需求的变化，您可能需要更改 Microsoft 365 计划。 您可以从当前订阅切换到另一个订阅： 
  
- **在同一服务系列中：** 例如，您可以从 Microsoft 365 商业版移动到 Microsoft 365 商业标准，或从 Office 365 E1 移动到 Office 365 E3。
- **从独立计划：** 例如，您可以从 Exchange Online 计划1移动到 Office 365 E1。     
- **到不同的服务系列：** 例如，您可以从 Microsoft 365 业务基本版迁移到 Office 365 E3。     
有关如何更改订阅（包括如何从试用版迁移到付费订阅）的信息，请参阅 [切换到其他 Microsoft 365 计划或订阅](https://support.office.com/article/HA104031833)。
  
可以在单个帐户中结合使用 Enterprise、Business 和独立计划（例如，Exchange Online 计划 1）。 但是，不会更改对每个计划席位数的现有限制。 例如，在 Microsoft 365 商业基本版和 Microsoft 365 商业标准中，每个计划最多可以有300个座位，但 Exchange Online 计划1上的用户数量不受限制。
