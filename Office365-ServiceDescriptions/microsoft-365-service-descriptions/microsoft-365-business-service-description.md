# <a name="microsoft-365-business-premium-service-description"></a>Microsoft 365 商业高级版服务说明

Microsoft 365 商业高级版是一个集成的解决方案，将 Microsoft 365 与高级安全性和设备管理结合在一起，从而帮助保护您的业务不受高级威胁。 此服务说明介绍了计划选项和功能。 有关 Office 365 的详细信息，请参阅[Microsoft 365 和 Office 365 服务说明](../office-365-service-descriptions-technet-library.md)。
  
## <a name="feature-availability-across-microsoft-365-business-standard-and-microsoft-365-business-premium-plans"></a>跨 Microsoft 365 商业标准和 Microsoft 365 商业高级版计划的功能可用性

Microsoft 365 商业高级版基本上包括 Microsoft 365 商业标准中提供的所有功能以及其他 Microsoft 安全功能。

### <a name="notes"></a>注释
<sup>1</sup>包括 Exchange Online 计划1。<br/>
<sup>2</sup> Azure 信息保护（AIP）启用受支持的信息权限管理（IRM）功能。 AIP Plan 1 包含在 Microsoft 365 商业高级版中。 <br/>
<sup>3</sup> Microsoft 365 商业高级版包括 Exchange Online 存档订阅。 每个 Exchange Online Archiving 订阅者最初都会获得 100 GB 的存档邮箱存储空间。 启用自动扩展存档后，只要达到 100 GB 存储容量，就会自动添加额外存储空间。 有关详细信息，请参阅[无限存档概述](https://docs.microsoft.com/office365/securitycompliance/unlimited-archiving)。 <br/>
<sup>4</sup>适用于 [https://securescore.office.com](https://securescore.office.com) 。 必须拥有管理员权限。 有关详细信息，请参阅[Microsoft 安全分数](https://docs.microsoft.com/office365/securitycompliance/microsoft-secure-score)。<br/>
<sup>5</sup>简化了 Windows 10 设备管理，内置在 Microsoft 365 管理中心。 "设备管理" 管理中心中内置的 iOS、Android、macOS 和其他跨平台设备管理的 Intune 管理功能，也授权与 Microsoft 365 商业高级版配合使用。 此外，还可以在设备管理管理中心内管理与 WiFi 配置文件和 VPN 证书等内容的第三方应用程序和配置。 <br/>
<sup>6</sup>此计划附带 Microsoft Stream plan 2，其中包括视频门户、语音到文本对话、深度搜索和人脸检测。<br/>
<sup>7</sup> Microsoft 365 提供了 Office 365 高级威胁防护计划1。 有关详细信息，请参阅[Microsoft 365 商业高级安全性和合规性功能](https://docs.microsoft.com/microsoft-365/business/security-features)。<br/>
<sup>8</sup> Microsoft 365 商业高级版包括以下 Azure AD 高级 P1 功能：使用 AD 写回、Azure 多重身份验证（MFA）和条件访问的自助服务密码重置。 它不包括完整的 Azure AD 高级 P1。</br>
<sup>9</sup> Microsoft 365 商业高级订阅者许可使用适用于 IOS、Android、MacOS 和其他跨平台设备管理的完整 Intune 功能。 在 Microsoft 365 商业高级版的管理中心中不可用的功能，如第三方应用管理和 WiFi 配置文件的配置、VPN 证书等。可以在完整 Intune 控制台中管理。</br>
<sup>10</sup>如果你有适用于 windows 7 和8.1 专业人员的设备，Microsoft 365 商业高级版将提供到 Windows 10 专业版的升级。 使用 Microsoft 365 商业高级版不仅可以获取[Windows Pro 中已启用的 Microsoft Defender 安全功能](https://go.microsoft.com/fwlink/p/?linkid=2089903)，还可以访问高级功能（如基于 web 的威胁的网络保护）以及主机入侵防护规则来缓解勒索软件。 设置 Microsoft 365 商业高级版帐户时，请选择 "应用保护" 和 "设备配置设置"。 这些设置决定了在设备加入 Azure Active Directory 时自动应用于设备的策略。</br>
<sup>11</sup>完整的 Azure AD 高级 P1 即将向 Microsoft 365 商业高级客户推出。 有关详细信息，请参阅[此博客文章](https://aka.ms/aadp1smbblog)。
