# <a name="microsoft-365-business-premium-service-description"></a>Microsoft 365 商业高级版服务说明

Microsoft 365 商业高级版是一个集成解决方案，将 Microsoft 365 一流的工作效率与高级安全和设备管理汇集在一起，以帮助保护你的企业防御高级威胁。 此服务说明介绍了计划选项和功能。 有关 Office 365 的信息，请参阅 [Microsoft 365 和 Office 365 服务说明](../office-365-service-descriptions-technet-library.md)。

### <a name="notes"></a>注释
<sup>1</sup> 包括 Exchange Online 计划 1。<br/>
<sup>2</sup> Azure 信息 (AIP) 支持的信息权限管理 (IRM) 功能。 AIP 计划 1 包含在 Microsoft 365 商业高级版中。 <br/>
<sup>3</sup> Microsoft 365 商业高级版包括Exchange Online Archiving订阅。 每个 Exchange Online Archiving 订阅者最初都会获得 100 GB 的存档邮箱存储空间。 启用自动扩展存档后，只要达到 100 GB 存储容量，就会自动添加额外存储空间。 有关详细信息，请参阅 [无限制存档概述](/office365/securitycompliance/unlimited-archiving)。 <br/>
<sup>4</sup> 在 [https://securescore.office.com](https://securescore.office.com) 中可用。 必须拥有管理员权限。 有关详细信息，请参阅 [Microsoft 安全分数](/office365/securitycompliance/microsoft-secure-score)。<br/>
<sup>5</sup> 内置于 Microsoft 365 管理中心的简化 Windows 10 设备管理。 内置于设备管理管理中心的 iOS、Android、macOS 和其他跨平台设备管理的 Intune 管理功能，还获得与 Microsoft 365 商业高级版一同使用的许可。 也可在设备管理管理中心管理第三方应用和 WiFi 配置文件和 VPN 证书等内容的配置。 <br/>
<sup>6</sup> 此计划随附 Microsoft Stream 计划 2，其中包括视频门户、语音到文本转录、深度搜索和人脸检测。<br/>
<sup>7</sup> Microsoft 365 附带 Microsoft Defender for Office 365 计划 1。 有关详细信息，请参阅 [Microsoft 365 商业高级版安全性和合规性功能](/microsoft-365/business/security-features)。<br/>
<sup>8</sup> Microsoft 365 商业高级版订阅者已获得许可，可使用适用于 iOS、Android、MacOS 和其他跨平台设备管理的完整 Intune 功能。 Microsoft 365 商业高级版管理中心中不可用的功能（如第三方应用管理和 WiFi 配置文件的配置、VPN 证书等）可以在完整的 Intune 控制台中进行管理。</br>
<sup>9</sup> 如果你的设备已获得 Windows 7 和 8.1 专业版许可，Microsoft 365 商业高级版将提供到 Windows 10 专业版升级。 但是，此许可证并不涵盖 Windows 10 专业版的 Windows 7.1 或 8 家庭版。 借助 Microsoft 365 商业高级版，你不仅获得 Windows 专业版中启用的 [Microsoft Defender](https://wfbdevicemanagementprod.blob.core.windows.net/windowsforbusiness/Windows10_CommercialEdition_Comparison.pdf)安全功能，还可以访问基于 Web 的威胁的网络保护以及主机入侵防护规则以减少勒索软件等高级功能。 设置 Microsoft 365 商业高级版帐户时，选择应用保护和设备配置设置。 这些设置确定在设备加入 Azure Active Directory 时自动应用的策略。</br>
