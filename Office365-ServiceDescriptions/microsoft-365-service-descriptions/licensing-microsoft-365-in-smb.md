# <a name="licensing-microsoft-365-for-small-and-medium-sized-businesses"></a>Microsoft 365 针对中小型企业的许可

Microsoft 365 商业高级版是适用于中小型商业版或 SMB 客户 (一) 产品。 Microsoft 365 商业高级版是一个集成解决方案，将 Microsoft 365的工作效率与高级安全和设备 **管理整合在一起**。

本文比较了 SMB 的许可选项，并着重介绍了 Microsoft 365 商业高级版。
