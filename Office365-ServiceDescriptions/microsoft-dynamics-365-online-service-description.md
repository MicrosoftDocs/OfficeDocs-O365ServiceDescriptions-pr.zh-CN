# <a name="microsoft-dynamics-365-online-service-description"></a>Microsoft Dynamics 365（在线版）服务说明

从年 12 月 2016年更新 Dynamics 365 (在线和本地)，Dynamics CRM 功能现在已包含作为 Dynamics 365，一套智能业务应用程序的一部分。动态 365 统一客户关系管理 (CRM) 和企业资源规划 (ERP) 功能通过提供新的应用程序来帮助管理特定的业务功能。 
  
这些应用程序包括：
  
- Dynamics 365 for Sales
    
- Dynamics 365 for Customer Service
    
- Dynamics 365 for Field Service
    
- Dynamics 365 for Project Service Automation
    
- Dynamics 365 for Operations
    
- Dynamics 365 财务
    
若要详细了解 Dynamics 365 功能，请访问 [Microsoft Dynamics 365](https://www.microsoft.com/en-us/dynamics365/home)。 
  
## <a name="dynamics-365-plans"></a>Dynamics 365 计划

Dynamics 365 具有两个版本： 企业和业务。这些两个版本内有多个计划可用专门符合使用模式和大型和小型企业的需求。应用程序分别是也可用。 
  
若要详细了解 Dynamics 365 功能，请访问 [Microsoft Dynamics 365](https://www.microsoft.com/en-us/dynamics365/home)。
